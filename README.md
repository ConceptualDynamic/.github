# Conceptual Dynamic GitHub Defaults

This repository contains default contribution guidance for the Conceptual Dynamic GitHub organization.

Main files:

- `.github/pull_request_template.md`: default pull request template for repositories that do not define their own template.
- `PULL_REQUEST_TEMPLATE.md`: root copy of the default pull request template for compatibility.
- `CONTRIBUTING.md`: contribution guidance for human and AI-assisted work.
- `CLAUDE.md`: context for Claude when working with repositories in `conceptualdynamic`.
- `.github/workflows/validate-ai-attribution.yml`: validation workflow template installed per repository to make attribution mandatory.
